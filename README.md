# AI-Project
the game uses animals of different sizes (Chick 🐤, Cat 🐱, Fox 🦊, Panda 🐼, Elephant 🐘) that must be stacked correctly according to size constraints. The goal is to move all animals from the starting peg (Peg 0) to the goal peg (Peg 2) following the rule: *a larger animal cannot be placed on top of a smaller animal.
